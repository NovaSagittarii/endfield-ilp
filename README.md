# Endfield PAC Optimizer

Maximizes some objective function (configured for Regional Development) for
factory output while abiding by power and ore constraints.

Made for fun since it's an silly application of linear programming.

![Screenshot](https://github.com/user-attachments/assets/46b7dc05-ed56-4095-8118-0d930d8f224b)

## Running

Install Python and install uv.

```sh
uv sync
uv run streamlit run main.py
```
